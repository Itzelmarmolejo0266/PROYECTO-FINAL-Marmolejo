# PROYECTO-FINAL-Marmolejo

## 1. datos_ejemplo.csv

Contiene información básica de productos, incluyendo:

- **id**: Identificador numérico del producto.
- **producto**: Nombre del producto registrado.
- **precio**: Costo unitario del producto.

### Ejemplo de datos:

| id | producto | precio |
|----|----------|--------|
| 1  | Laptop   | 750.00 |
| 2  | Mouse    | 15.50  |
| 3  | Teclado  | 30.00  |

---

## 2. clientes.csv

Incluye los datos de clientes registrados en el sistema:

- **id_cliente**: Identificador único del cliente.
- **nombre**: Nombre completo de la persona.
- **correo**: Correo electrónico del cliente.

### Ejemplo de datos:

| id_cliente | nombre       | correo         |
|------------|--------------|----------------|
| 101        | Ana Pérez    | ana@gmail.com  |
| 102        | Luis Torres  | luis@gmail.com |
| 103        | María López  | maria@gmail.com |

---

## Propósito

Estos archivos sirven como bases de datos iniciales para sistemas o prácticas de manejo de archivos CSV, permitiendo pruebas de lectura, filtrado, análisis o carga en proyectos de software.
